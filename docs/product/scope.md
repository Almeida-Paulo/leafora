# Product Scope

Leafora is built as a production-oriented platform from the beginning. The first
release is limited in surface area, but it uses the same domain boundaries
intended for later releases.

## Current Product Surface

- ecological project marketplace;
- project detail pages;
- Sui wallet connection;
- support tiers with supporter NFTs;
- allocation points;
- public evidence registry;
- browser-side evidence hash preparation;
- Sui Move package for project/vault/support/evidence objects;
- backend API for project, evidence and support records.

## Product Guardrails

- support, not guaranteed investment return;
- allocation points, not APY;
- conditional future claims only after actual revenue deposits;
- evidence registry, not certified carbon credit issuance;
- curated project launch before third-party project creation opens.

## Release Progression

### Public Devnet Release

- deploy static dApp;
- deploy backend API;
- publish Sui Move package to devnet;
- create curated pilot projects;
- index support and evidence events.

### Controlled Mainnet Release

- complete contract tests;
- complete backend migrations;
- complete security review;
- use production RPC/indexer;
- publish only curated internal projects.

### Creator Platform

- creator accounts;
- project submission workflow;
- validator/admin queue;
- evidence upload app;
- KYB/KYC and compliance workflows where required.
